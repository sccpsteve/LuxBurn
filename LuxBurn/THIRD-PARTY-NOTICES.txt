LuxBurn Third-Party Notices
===========================

LuxBurn includes third-party components to provide optical media workflows.

cdrtools / Cygwin runtime files
-------------------------------

Bundled files:

- Tools\cdrtools\cdrecord.exe
- Tools\cdrtools\mkisofs.exe
- Tools\cdrtools\readcd.exe
- Tools\cdrtools\cdda2wav.exe
- Tools\cdrtools\isoinfo.exe
- Tools\cdrtools\share\lib\siconv\*
- Tools\cygwin\cygwin1.dll
- Tools\cygwin\sh.exe

These files are used as external command-line tools for disc writing, ISO
creation support, disc reading, and runtime compatibility.

Project/source references:

- cdrtools: https://sourceforge.net/projects/cdrtools/
- Cygwin: https://www.cygwin.com/

The bundled native tools retain their own copyrights and licenses. If you
redistribute LuxBurn publicly, verify that the bundled binaries, matching
source code, and license/source-offer requirements are satisfied for the exact
binary builds being distributed.

User-provided Segoe TV font package
-----------------------------------

Bundled files:

- Assets\Fonts\SegoeTVRegular.TTF
- Assets\Fonts\SegoeTVItalic.TTF
- Assets\Fonts\SegoeTVBold.TTF
- Assets\Fonts\SegoeTVBoldIt.TTF

LuxBurn loads these fonts privately at runtime and falls back to Tahoma if the
font files are missing or cannot be loaded.

The font package was supplied as a local project asset. Before public
redistribution, confirm that the font license permits bundling and redistribution
with this application.
